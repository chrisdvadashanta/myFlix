myFlix

To build the server-side component of a “movies” web application.
The web application will provide users with access to information
about different movies, directors, and genres. Users will be able
to sign up, update their personal information, and create a list
of their favorite movies.

Nr. Business Logic URL HTTTP Method Request body data format Response body data format
1. Return a list of ALL movies to the user /movies GET None A JSON Object holding data about all the movies
2. Return data (description, genre, director, image URL,
whether it’s featured or not) about a single movie by title to the user		 /movies/movie[description][genre][director][image][URL] GET None A JSON Object holding data about all the movies
3. Return data about a director (bio, birth year, death year) by name /movies/director/[bio][birthyear][deathyear] GET None A JSON Object holding data about one director
4. Return data about a genre (description) by name/title (e.g., “Thriller”) /genre[name][description] GET None A JSON Object holding data about one genre
